# linksockslib (gopy backend)

This directory contains the standalone `linksockslib` Python package.

- `linksockslib` is the gopy/CPython-extension backend.
- `linksocks` (in `../python/`) is the default package and ships the FFI backend.

Build/install locally:

- `make python-gopy-install`
- `make python-gopy-test`
